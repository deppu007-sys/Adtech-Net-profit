# Adtech-Net-profit
# 📊 AdTech Net Profit Margin Calculator

## 🔍 Overview
This project calculates net profit and net profit margin for advertising campaigns.

## 📐 Formula
Net Profit = Revenue - Cost - Expenses  
Net Profit Margin = (Net Profit / Revenue) * 100

## 🚀 Features
- Multi-campaign analysis
- Includes operational expenses
- Simple Python implementation

## 🛠 Tech Stack
- Python

## 📈 Use Case
Helps AdTech companies understand actual profitability after expenses.
